# KPI Monitor ML

## Тема ВКР

**Разработка системы мониторинга и анализа KPI сотрудников организации с использованием методов машинного обучения.**

Лянгинен Илья Борисович

## Назначение проекта

KPI Monitor ML — это десктопное приложение на Python для мониторинга, анализа и прогнозирования KPI сотрудников организации.

Система позволяет:

- вести учет сотрудников, отделов и должностей;
- хранить KPI-показатели и KPI-записи;
- разграничивать доступ по ролям пользователей;
- анализировать KPI по сотрудникам и отделам;
- обучать ML-модель для прогнозирования риска снижения KPI;
- формировать отчеты в форматах DOCX и XLSX;
- вести журнал действий пользователей;
- выполнять резервное копирование базы данных.

## Основные роли пользователей

В системе предусмотрены 3 роли:

1. **Администратор**
   - полный доступ к системе;
   - управление пользователями;
   - доступ к настройкам;
   - обучение ML-модели;
   - просмотр журнала действий.

2. **Руководитель**
   - просмотр сотрудников своего отдела;
   - работа с KPI-записями;
   - просмотр аналитики отдела;
   - формирование отчетов.

3. **Сотрудник**
   - просмотр личного кабинета;
   - просмотр собственных KPI;
   - просмотр персонального ML-прогноза;
   - доступ к справке.
