# KPI Monitor ML

## Тема ВКР

Разработка системы мониторинга и анализа KPI сотрудников организации с использованием методов машинного обучения.

## Назначение проекта

Проект представляет собой прототип программной системы для учета, мониторинга и анализа KPI сотрудников организации.

Планируемые функции:

- учет сотрудников;
- учет отделов и должностей;
- хранение KPI-показателей;
- анализ динамики KPI;
- прогнозирование риска снижения эффективности сотрудника;
- разграничение прав доступа по ролям;
- формирование отчетов в форматах DOCX и XLSX;
- сохранение и загрузка обученной ML-модели.

## Планируемый стек технологий

- Python
- PySide6
- SQLite / SQLAlchemy
- pandas
- scikit-learn
- matplotlib
- python-docx
- openpyxl
